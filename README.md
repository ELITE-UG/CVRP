# CVRP (Capacitated Vehicle Routing Problem)
This program is a web-based Distribution Route Optimization System developed using Python and Streamlit. It is designed to optimize vehicle delivery routes by solving the Capacitated Vehicle Routing Problem with Time Windows (CVRPTW). The application helps logistics planners determine the most efficient delivery schedule while considering vehicle capacity, customer demand, travel distance, and operational costs.
